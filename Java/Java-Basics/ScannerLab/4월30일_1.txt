1. 클래스명 : ScannerLab1 
2. 표준입력으로 숫자 두 개와 연산자 문자 한 개를 입력받아서
    각각 변수에 저장한다.
3. 입력된 연산자가 '+' 이면 입력된 두 개 숫자의 덧셈을 처리한다.
    입력된 연산자가 '-' 이면 입력된 두 개 숫자의 뺄셈을 처리한다.
    입력된 연산자가 '/' 이면 입력된 두 개 숫자의 나눗셈을 처리한다.
    입력된 연산자가 '*' 이면 입력된 두 개 숫자의 곱셈을 처리한다.
4. 결과는 다음과 같이 출력한다.
    
    XX 와 YY의 X 연산 결과는 ZZ 입니다.
