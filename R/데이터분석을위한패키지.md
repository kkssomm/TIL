# 데이터 분석을 위한 패키지



## reshape2 패키지

- melt() 함수 옵션
  - data:melt() 함수로 변형할 데이터 세트 입력
  - na.rm = FALSE : 결측 값 제외
  - value.name="value" : 행으로 바꾸고 싶은 열 이름

- melt(데이터 세트, id.var="기준 열",measure.vars="변환 열")

```
melt_test2 <-melt(airquality, id.vars=c("month","wind"),measure.vars="ozone")
head(melt_test2)
```
